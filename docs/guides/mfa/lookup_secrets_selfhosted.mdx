id: lookup_secrets_selfhosted
title: Configuration in Self-Hosted Instances
---

When working with self-hosted instances of the Ory Identity Service (Kratos), add the
`lookup_secret` method to `selfservice/methods` in the configuration file to enable Lookup Secrets: 

```yaml title="kratos.config.yml"
selfservice:
    methods:
      lookup_secret:
        enabled: true
```
