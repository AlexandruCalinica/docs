---
id: lookup_secrets_cli
title: Configuration in Ory CLI
---

Follow these steps to enable Lookup Secrets: 

1. Get the Identity Service configuration from your project 
and save it to a file: 

  ```shell
  ## List all available projects
  ory list projects

  ## Get config
  ory get identity-config <project-id> --format yaml > identity-config.yaml
  ```

2. Find `lookup_secret` in `selfservice/methods` and set `enabled` to `true`:

  ```yaml
  lookup_secret:
      enabled: true
  ```

3. Update the Ory Cloud Identity Service configuration using the file you worked
   with:

  ```shell
  ory update identity-config <project-id> --file updated_config.yaml
  ```
